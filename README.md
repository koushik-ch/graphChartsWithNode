# Representing historical football world cup data in graphs using Node.js

## DemoVideo

[Youtube](https://www.youtube.com/watch?v=tTAs9kx7rRs)

## Project setup
```
npm install
```

### Compilation
```
node app.js
```

### Now the server is up and listening at port 3000
``` 
upload the worldcups.csv file
```
